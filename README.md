# Fake Job Posting Detection System
## Project Overview

Online job portals often contain fraudulent job postings that mislead job seekers.  
This project builds an end-to-end system to detect fake job postings using:

- SQL for data storage and analysis
- Python (NLP + Machine Learning) for fraud detection
- Power BI for interactive fraud analytics dashboard

The machine learning model achieved **96% accuracy**.

---

## Project Architecture

Dataset → SQL Database → Python ML Model → Predictions → Power BI Dashboard

---

## Technologies Used

- Python
- Pandas
- Scikit-learn
- TF-IDF
- Logistic Regression
- MySQL
- SQLAlchemy
- Power BI

---

## Machine Learning Model

The model uses:

- **Logistic Regression Classifier**

Steps performed:

1. Data cleaning
2. Text feature engineering
3. Train-test split
4. Model training
5. Evaluation

Model accuracy: **96%**

---

## SQL Analysis

SQL was used to explore fraud patterns.

Example queries include:

- Fake vs Real job distribution
- Fraud percentage
- Fake jobs by location
- Missing salary analysis

---

## Power BI Dashboard

The Power BI dashboard provides business insights such as:

- Total jobs and fraud percentage
- Fake vs real job distribution
- Fraud by location
- Salary information analysis

---

## Key Insights

- Fraudulent jobs represent a small percentage of total postings
- Jobs without salary information show higher fraud risk
- Certain locations contain higher concentration of fake jobs
